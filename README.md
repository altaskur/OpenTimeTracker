## ⚡ Inicio Rápido

```bash
# Instalar dependencias
npm install

# Desarrollo con Electron (recomendado)
npm run dev

# Solo Angular en navegador
npm start

# Build completo (producción)
npm run build

# Ejecutar Electron (requiere build previo)
npm run electron

# Tests
npm test
```
